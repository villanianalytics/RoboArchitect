RoboArchitect

Description

RoboArchitect is a command-line application that can help you with zipping and unzipping files, convert from .csv file to .xlsx file and vice versa, send an email, encrypt a password and write it to a file, send a request to specified url with a username and a password and get a response, parse data from JSON, and run SQL against a .csv/.txt file or create SQLite database and run SQL request there.

Zip and Unzip: Archives files and directories in a .zip or decompresses .zip files into the specified directory.

Conversion: Converts a .csv file to a .xlsx file, and can also perform the inverse conversion.

Connect and Parse: Sends a request for a web service and receives a response from it.

Encrypt: Encrypts your password and you can use it in commands where a password is required.

Email: Sends emails to any users and supports all standard functions.

Query Delim: Executes SQL queries using delimited files as the basis. The file name is the table name, and the first line is the table headers, everything else is the table data.

SQL: Creates an SQLite database, creates a table with data from a delimited file there or adds data to an existing table. It can also execute SQL queries to the database and write the result to a file or display it on the console.

Zip

Archives file or directory and save it in a specified directory.

To make a .zip file you need to specify /srcDir and /destFile attributes.

For /srcDir, you could pass either directory or file.

For /destFile, you should pass an expected .zip file. If the directory of this file doesn't exist it would create it.

Unzip

Unzip .zip file to a specified directory.

To unzip an archive you should specify /srcFile and /destFile attributes.

For /srcFile, you should pass either local or global .zip file path.

For /destFile, you should pass a directory where all your unzipped files will be saved.

Convert

Converts .csv file to .xlsx file.

To convert .csv files to .xlsx you should specify /srcFile, /destFile, /delim and /sheetname attributes.

For /srcFile, you should pass either local or global .csv file path.

For /destFile, you should pass a directory with a file where you want to save your .xlsx file.

For /delim, you should pass a delimiter of .csv file, usually, it is "," or ";".

For /sheetname, you should pass a .xlsx sheet name.

Password

Encrypts user password.

For /passwordFile, you should pass a file where your encrypted file will be saved.

Connect

Connect to specified url with required username and password, then get a response and save it to a file.

For /destFile, you could pass a file where a response would be saved.

For /config, you should pass a config file.

For /jsonPath, you could pass a JSON filter pattern.

The config file should have user and url fields. If you wrote the password field there you may not use /passwordFile method.

Here is one of the possible realizations:

#Authentication Properties

url=url

user=username

password=password

httpmethod=GET

Json Path

Json path is needed to filter a JSON file and take from there just a needed part. It takes the needful part of a JSON by a pattern.

For /srcFile, you should pass a file with JSON inside.

For /jsonPath, you should pass a JSON filter pattern.

For /destFile, you could pass a file where the filtered result will be saved.

Email

You could send an email with or without an attached file to another person.

For /config, you should pass a config file.

If you want to use an encrypted password from a file you could path it to /passwordFile else you should pass it to a config file.

Here is one of the possible realizations of the email config file:

#Email Properties

fromaddress=email@gmail.com

# If you do not have an encrypted password file.
password=password

toaddress=emailTo@gmail.com

# Optional.
ccaddress=copyEmail@xrpmail.com

# Optional.
bccaddress=sicretCopyEmail@enayu.com

subject=Here must be a subject of a letter.

# If you want to send a message as HTML text you should pass Y else N.
htmlbody=Y

body=<h1> Here could be a text. </h1>

# Optional.
attachment=file.txt

ctype=TLS

smtpserver=smtp.gmail.com

port=587
# SSL -> port 465; TLS -> port 587.

Query Delim

Allows a user to run SQL against a .csv/.txt file.

For /srcFile, .txt or .csv file with data.

For /delim, file delimiter.

For /query, SQL request.

For /destFile, destination file.

For /suppressHeaders, specify whether headers should be suppressed.

For /skipLines, number of lines in file to skip

For /skipDataLines, number of data lines after the header to skip.

SQL

Allows a user to create SQLite database, create table or add data to it by using file with delimiter.

For /connection, specify the jdbc connection. Note: For SQLite, just enter "SQLITE"

For /op, create database option, import table to database or run query on database.

For /db, SQLite database destination should ends by .db.

For /srcFile, .txt or .csv file with data.

For /table, SQLite database table name.

For /delim, source file delimiter.

For /mode, OVERWRITE or APPEND database table.

For /query, SQL request from console or a file.

For /return, true if output should be printed, else false.

For /destFile, destination file.

For /header, true if headers of database table should be skipped, else they would be printed.

Examples

Zip

Archive file file.txt from the example/manual directory to the manual.zip file in the current directory.

RoboArchitect -zip /srcDir="example/manual/file.txt" /destFile="manual.zip"

Archive file file.txt from the example/manual directory to the manual.zip file in the current directory with logging to the logs.txt file.

RoboArchitect -zip /srcDir="example/manual/file.txt" /destFile="manual.zip" /log="logs.txt"

Archive directory example from the current directory to the directory.zip file from the zipped/directory directory.

RoboArchitect -zip /srcDir="example" /destFile="zipped/directory/manual.zip"

Archive directory example from thr current directory to the directory.zip file from the zipped/directory directory with logging to the logs.txt file.

RoboArchitect -zip /srcDir="example" /destFile="zipped/directory/manual.zip" /log="logs.txt"
Unzip

Unzip manual.zip file from the unzipped/files directory to the current directory.

RoboArchitect -unzip /srcFile="unzipped/files/manual.zip" /destDir=""

Unzip manual.zip file from the zipped/files directory to the dest/files directory.

RoboArchitect -unzip /srcFile="zipped/files/manual.zip" /destDir="dest/files"

Unzip manual.zip file from the unzipped/files directory to the current directory with logging to the logs.txt file.

RoboArchitect -unzip /srcFile="unzipped/files/manual.zip" /destDir="" /log="logs.txt"

Unzip manual.zip file from the zipped/files directory to the dest/files directory with logging to the logs.txt file.

RoboArchitect -unzip /srcFile="zipped/files/manual.zip" /destDir="dest/files" /log="logs.txt"
Convert

Convert manual.csv file from the current directory, put it to the test.xlsx file from the dest/files directory by using ; .csv delimiter, and rename a .xlsx file sheet to the ABCD.

RoboArchitect -convert /srcFile="manual.csv" /destFile="dest/files/test.xlsx" /delim=";" /sheetname="ABCD" /log="logs.txt"

Convert manual.csv file from the current directory, put it to the test.xlsx file from the dest/files directory by using ; .csv delimiter, and rename a .xlsx file sheet to the ABCD and write all logs to the logs.txt file. 

RoboArchitect -convert /srcFile="manual.csv" /destFile="dest/files/test.xlsx" /delim=";" /sheetname="ABCD" /log="logs.txt"

Convert test.xlsx file from the converted directory to the test.csv file from the converted directory by ABCD sheetname and write all logs to the app.log file. 

RoboArchitect -convert /srcFile="converted/test.xlsx" /destFile="converted/test.csv" /sheetname="ABCD" /log="app.log"

Convert test.xlsx file from the converted directory to the test.csv file from the converted directory by using ; .csv delimiter and ABCD sheetname. 

RoboArchitect -convert /srcFile="converted/test.xlsx" /destFile="converted/delims/test.csv" /delim=";" /sheetname="ABCD"
Encrypt

Encrypt your entered password and save it to the password.txt from the current directory.

RoboArchitect -password /passwordFile="password.txt"

Encrypt your entered password and save it to the password.txt from the current directory and write all logs to the logs.txt file.

RoboArchitect -password /passwordFile="password.txt" /log="logs.txt"
Connect

Connect to a web service by using the config.properties file from the current directory and the password.txt file with encrypted password, save results to the response.json from the current directory.

RoboArchitect -connect /destFile="response.json" /config="config.properties" /passwordFile="password.txt"

Connect to a web service by using the config.properties file from the current directory and the password.txt file with encrypted password, save results to the response.json from the current directory and write all logs to the logs.txt file.

RoboArchitect -connect /destFile="response.json" /config="config.properties" /passwordFile="password.txt" /log="logs.txt"

Connect to a web service by using the config.properties file from the current directory and save results to the response.json from the current directory.

RoboArchitect -connect /destFile="response.json" /config="config.properties"

Connect to a web service by using the config.properties file from the current directory and save results to the response.json from the current directory and write all logs to the logs.txt file.

RoboArchitect -connect /destFile="response.json" /config="config.properties" /log="logs.txt"

Connect to a web service by using the config.properties file from the current directory.

RoboArchitect -connect /config="config.properties"

Connect to a web service by using the config.properties file from the current directory.

RoboArchitect -connect /config="config.properties" /reqType=POST /body="BODY TEXT\n IS HERE" /mediaType=multipart/form-data /srcFile="target/smile.png" /destFile=output.txt

Connect to a web service by using the config.properties file from the current directory, GET HTTP connection method, and by using password from encrypted file.

RoboArchitect -connect /reqType=GET /config="connect/config.properties" /passwordFile="password/connectPassword.txt"

Filter a json from connect method by using the config.properties file from the current directory and the id pattern, then save all results to the response.json file from the current directory.

RoboArchitect -connect /destFile=response.json /config=config.properties /jsonPath=$.items[?(@.name==\"NAME\"||@.name==\"NAME\")].id

RoboArchitect -jsonpath /srcFile=response.json /jsonPath=$.items[?(@.name==\"NAME\"||@.name==\"NAME\")].id /destFile=filtered.json

Filter a json from the response.json file by using the id pattern.

RoboArchitect -jsonpath /srcFile=response.json /jsonPath=$.items[?(@.name==\"NAME\"||@.name==\"NAME\")].id
Email

Send an email by using the email.properties file and password.txt file with encrypted password.

RoboArchitect -email /config=email.properties /passwordFile=password.txt

Send an email by using the email.properties file.

RoboArchitect -email /config=email.properties

Send an email by using the email.properties file and save all logs to the logs.txt file. 

RoboArchitect -email /config=email.properties /passwordFile=password.txt /log=logs.txt

Send an email by using the email.properties file, subject of email and email body, save all logs to the logs.txt file. 

RoboArchitect -email /subject="SUBJECT" /body="Let's make next Line<br>Next Line" /config=email.properties /log=test.log

Send an email by using the email.properties file, subject of email and email body. 

RoboArchitect -email /subject="SUBJECT" /body="Hello world" /config=email.properties
Query Delim

Run SQL for the test.csv file in the converted directory with SQL request from a command line.

RoboArchitect -querydelim /srcFile="converted/test.csv" /query="Select * from test where \"3\"='c' or \"1\"='qwerty'"

Run SQL for the test.csv file in the converted directory with SQL request from a command line with , delimiter and write all logs to app.log file.

RoboArchitect -querydelim /srcFile="converted/test.csv" /query="Select * from test where \"3\"='c' or \"1\"='qwerty'" /delim="," /destFile="query/out.txt" /log="app.log"

Run SQL for the test.csv file in the converted directory with SQL request from a command line with default delimiter, skip two file's lines and write all logs to app.log file.

RoboArchitect -querydelim /srcFile="converted/test.csv" /query="Select * from test" /destFile="query/out.txt" /skipDataLines=2 /log="app.log"
SQL

Create empty SQLite store.db database.

RoboArchitect -sql /connection="SQLITE" /op=createDB /db=store.db

Create empty SQLite store.db database.

RoboArchitect -sql /connection="SQLITE" /op=createDB /db=store.db

Creating a table named ABCD in the store.db database and filling it with data from the test.csv file.

RoboArchitect -sql /connection="SQLITE" /op=importTable /db=store.db /mode=overwrite /srcFile=test.csv /table=ABCD

Add A table in SQLite store.db database by adding test2.csv file data.

RoboArchitect -sql /connection="SQLITE" /op=importTable /db=store.db /mode=overwrite /srcFile=test2.csv /table=A

To overwrite A table in SQLite store.db database by adding test3.csv file data.

RoboArchitect -sql /connection="SQLITE" /op=importTable /db=store.db /mode=overwrite /srcFile=test3.csv /table=A

To add data to A table in SQLite store.db database by adding test4.csv file data.

RoboArchitect -sql /connection="SQLITE" /op=importTable /db=store.db /mode=overwrite /srcFile=test4.csv /table=A

To run SQL query on A table in SQLite store.db database and print result with headers to a console.

RoboArchitect -sql /connection="SQLITE" /op=queryDB /db=store.db /query="SELECT * from A" /header=true

To run SQL query on A table in SQLite store.db database and print result without headers to a file.

RoboArchitect -sql /connection="SQLITE" /op=queryDB /db=store.db /query="SELECT * from A WHERE \"NAME\"=\"TOM\"" /destFile=file.txt

To run SQL query from a query.txt file on A table in SQLite store.db database and print result without headers to a console.

RoboArchitect -sql /connection="SQLITE" /op=queryDB /db=store.db /query="query.txt"
